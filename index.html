[Free performance with huge pages]
  • Imagine reducing cache misses and table walks without changing a line of code? Linux is somewhat stuck in the past and still using 4K page sizes but supports up to 1GB. You got all that ram, your CPU cache is growing with each generation, so let's make use of that and see some scaled performance benefits.
